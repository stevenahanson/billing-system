# billing-system

## Getting Started

First, clone the repo:

```bash
git clone https://github.com/stevenahanson/billing-system.git
```

Install packages:

```bash
npm install
# or
yarn add
```

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
